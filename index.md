# 🧠 LLM Wiki 知识地图

> "一次整理、持续复用、复利增长" — 基于 Karpathy llm-wiki 模式

## 核心理念

- **传统 RAG**：每次查询都重新检索碎片信息，无积累
- **LLM Wiki**：LLM 维护持久化知识，知识越用越丰富

**人机分工**：人类筛选资源、提问；LLM 整理、归档、交叉引用

## 📂 知识目录

| 类型 | 用途 |
|------|------|
| **summary** | 摘要页 - 对原始资源的高密度总结 |

| **concept** | 概念页 - 核心理论与定义 |
| **comparison** | 对比页 - 多个选项的对比分析 |
| **overview** | 概览页 - 某个领域的全景图 |
| **synthesis** | 综合页 - 整合多个来源的核心洞察 |
